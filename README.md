> **TruIntel AI** is a production-grade AI SaaS platform designed to detect deceptive, AI-generated, or paid product reviews on platforms like Amazon and Flipkart. Built entirely with Next.js 14 and custom NLP algorithms.

### 🔗 **[Explore the Live App: truintel-ai.vercel.app](https://truintel-ai.vercel.app)**

---

## 🚀 Why I Built This
As e-commerce grows, the volume of fake reviews drastically skews consumer trust. I wanted to build a real-time, client-side engine capable of dissecting review text using Natural Language Processing (NLP) heuristics without relying on expensive backend APIs. 

This project served as a deep dive into **advanced frontend architecture**, **complex state management**, and building a highly interactive **"Antigravity UI"** using Framer Motion and modern CSS glassmorphism.

---

## ✨ Features Engine

| Module | Capability Description |
|---------|-------------|
| 🎯 **Trust Score Engine** | Calculates a 0–100 authenticity score dynamically weighting 15+ heuristic properties. |
| 🔍 **Deceptive Phrase Lexicon** | Scans inputs against a regex database of 200+ known deceptive pattern combinations. |
| 📈 **Sentiment Analysis** | Flags unnatural emotional extremities often present in paid reviews. |
| ✅ **Authenticity Radar** | Generates a 6-axis relational chart mapping genuine reviewer behavior signals. |
| ⚡ **Real-Time Execution** | Processes the entire algorithmic pipeline entirely within the client boundary under 2s. |
| 📊 **Custom Data Viz** | Renders dynamic, animated SVG charts completely from scratch (no external charting libs). |

## 🛠️ Technical Architecture

<details>
<summary><b>Click to view Tech Stack Details</b></summary>

- **Framework**: `Next.js 14` (App Router)
- **Language**: `TypeScript` (Strict typings across all heuristics and component boundaries)
- **Styling**: `Tailwind CSS` (Custom tokens, gradients, and glassmorphic UI elements)
- **Motion & Physics**: `Framer Motion` (Complex orchestration of component mounting and micro-interactions)
- **3D Graphics**: `Three.js` (Web-GL rendered particle background systems)
- **Deployment**: `Vercel` (Edge caching and CI/CD pipelines)

</details>

## 🖥️ Run It Locally Workspace

To spin up a local instance for development or testing:

```bash
# 1. Clone the repository
git clone https://github.com/harshal-nanekar11/truintel-ai.git

# 2. Navigate to your workspace
cd truintel-ai

# 3. Install dependencies
npm install

# 4. Start the Turbopack dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

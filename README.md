### 🔗 **[Live Demo: truintel-ai.vercel.app](https://truintel-ai.vercel.app)**

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎯 **Trust Score Engine** | Scores every review 0–100 using 15+ heuristic signals |
| 🔍 **Phrase Detection** | Matches against 200+ known fake-review patterns |
| 📈 **Sentiment Analysis** | Detects unnaturally extreme emotional tone |
| ✅ **Authenticity Signals** | 6-dimension scoring of genuine reviewer behaviour |
| ⚡ **Real-Time Analysis** | Full verdict in under 2 seconds, zero server calls |
| 📊 **Detailed Explanations** | Know exactly *why* a review was flagged |

---

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + custom glassmorphism
- **Animations**: Framer Motion
- **3D Graphics**: Three.js
- **Charts**: Custom SVG — no chart library needed
- **AI Logic**: Custom NLP heuristics (no paid APIs)

---

## 🚀 Run Locally

```bash
git clone https://github.com/harshal-nanekar11/truintel-ai.git
cd truintel-ai
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
